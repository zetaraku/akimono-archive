[トップ] > デザイン編 > スキンの活用
商人物語は，スキンという発想により，デザインの変更がしやすいよう設計されています。
ここでは，スキンを活用しながらデザインを変更する方法について解説します。
  • スキンを適用する
  • スキンを編集する
  • スキンで使った素材の著作権表示

△スキンを適用する

商人物語は，スキンという「デザインセット」を読み込んで，見栄えを作ります。
Windows でいうところの「デスクトップテーマ」のようなものをイメージしてもらえればいいと思います。

このため，スキンファイルを入手して取り替えるだけで，あっという間にデザインを変更することができます。
スキンファイルは，商人物語のダウンロードページで配布しております。
また，将来的には商人物語対応の素材サイトでも入手できるかもしれません。

スキンファイルを入手した後，それを適用してデザインを変更するには，ファイルをすべて，
ゲームフォルダ（watlas など）にそのままアップロードすればいいだけです。
その際，sindex.html などのファイルは，上書きすることになります。
スキンに専用のマニュアルが付属している場合には，そのマニュアルの指示に従ってください。
なお，マニュアルをアップロードする必要はありません。

△スキンを編集する

スキンファイルを書き換えることにより，お好みにより自由なデザインを作ることができます。
スキンの正体は，sindex.html などのＨＴＭＬファイルです。ファイルごとの役割は次の通りです。
・sindex.html … トップ画面のデザイン。
・smain.html … 通常画面のデザイン。
・sindex-mob.html … トップ画面のデザイン（携帯用）。
・smain-mob.html … 通常画面のデザイン（携帯用）。

これらのファイルを書き換えれば，デザインが変わってくれます。
デザインをまったく新しくする場合も，１から作るのではなく，今あるスキンファイルから改造していくようにしましょう。
※もっとも携帯用のデザインは，シンプルであることが必要なので，なるべく変えないほうがいいです。
ＨＴＭＬなので，書き換えはホームページを作るつもりでやればＯＫです。
JAVAを使おうがFlashをはりつけようが，ホームページでできることはそのまま反映されます。
ホームページビルダーやフロントページなどのホームページ編集ソフトもお使いいただけます。

ただし，以下の記号は，システム上特別な働きをします。削除してしまうと，その部分の表示が出なくなります。
・<!--SKINPAUSE-->	… プログラムの目印（２箇所ある）。削除してはならない。
・#SKINMENUTITLE#	… 今いる場所の名前に置き換えられる。
・#SKINMENU#		… 「次回決算まであと・・・」に置き換えられる。
・#SKINJOB#		… ジョブ画像に置き換えられる。
・#SKINMENUSUB#	… そこから行ける場所へのリンクに置き換えられる。
・#SKINCPU#		… サーバーの反応速度を示す数値に置き換えられる。
・#SKINTITLE#		… 街の正式名称に置き換えられる。
・#SKINNEW#		… 「新規店舗オープン」に置き換えられる。
・#SKINCOMMENT#	… $TITLE_COMMENT（_config.cgi）に置き換えられる。
・#SKINNAME#		… 以前ログインした名前が自動入力される。
・#SKINPW#		… 以前ログインしたパスワードが自動入力される。
・#SKINMAIL#		… 管理者メールアドレスに置き換えられる。
・#SKINVER#		… 商人物語のバージョンに置き換えられる。
・#SKINEDIT#		… 改造バージョンに置き換えられる。
※黒の太字は，トップ画面（sindex.html，sindex-mob.html）でのみ有効な記号です。
※それぞれの記号は，実際には有効な位置があり，大きく外れると無効になることがあります。

そのほか，スタイルシートの記述の中には，削除してしまうと大きく見栄えが崩れるものがあるので気をつけましょう。
いろいろ試行錯誤しながら，何となく理解していくのがよいと思います。

△スキンで使った素材の著作権表示

ホームページ素材サイトの素材を使ってデザインを作ったときに，著作権表示が必要になることがあります。
（たとえば，「素材使用の際はホームページのどこかにリンクをはることが条件」など。）
そのほか，あなたが作成したデザインについて，あなた自身の著作権を表示しておきたい場合もあるでしょう。

そこで，スキンファイルに関する著作権を，スタッフロールに簡単に追加できるようになっています。
ゲームフォルダ（watlas など）内の skin.pl をテキストエディタで編集すればＯＫです。
以下の編集例を参考にしてください。
＜編集例＞ … skin.pl 9行目から 
$disp.=<<"HTML";
<br><br>
<SPAN>背景素材</SPAN> ： ○○○ 様 ... <A HREF="http://アドレス/" target="_blank">名称</A>
<br><br>
<SPAN>スキン配布元</SPAN> ： ○○○ ... <A HREF="http://アドレス/" target="_blank">名称</A>
HTML
※ skin.pl は，スキンを配布するときにそれを添付することで，受け取った相手に著作権表示を渡すことができます。

著作権表示など，素材提供元が求める素材使用条件は必ず守るようにしてください。
見てないだろうと思うかもしれませんが，提供元にバレることは結構多いのです。
