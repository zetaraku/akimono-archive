■お礼と解説
　この度は『Nightfall』をご利用いただきありがとうございます。
　このスキンは、素材を他サイト様からお借りし、原作配布スキンをアレンジしたものです。
　再配布はご自由に行って構いませんが、背景・使用画像などの著作を変更することを禁止します。（背景・ロゴを完全に取り替えてしまう場合以外はこの限りではありません。が、これらの素材を一部でも使用する場合は著作表記は変えないでください。）

■スキンの設定方法
　設定方法は『商人物語』のマニュアルに従い、以下をアップロードしてください。

　skin.pl　　-------　著作が明記してあります
　smain-mob.html　----　原作無改造
　sindex-mob.html　---　原作無改造
　sindex.html　------　トップページ用スタイル
　smain.html　-------　その他ページ用スタイル
　mark.png　--------　テーブル背景画像
　star.gif　--------　降って来る星の画像
　bg.gif　----------　背景画像
　logo.gif ---------　『商人物語』ロゴ画像（オリジナルを使う場合は必要なし）
　image/　----------　item-type1～10.png（スキンと色揃いのもの）


　現在、水平線（hrタグ）の仕様は

　hr {filter: Alpha(Opacity=90, FinishOpacity=0, Style=2); height: 3px; color: #CC9933; border-style: dotted}

　となっていますが、シンプルなものをご希望の方は

hr {height: 1px; color: #666666}

　をお薦めします。
　ブラウザにより表示が出来ない場合がありますので、ご注意ください。
